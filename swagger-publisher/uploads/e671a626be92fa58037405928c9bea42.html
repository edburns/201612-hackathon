post /20170101/storeBackups
Create a new Etcd store backup. (createStoreBackup)
An asynchronous operation that creates an Etcd store backup.

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Request body

body (required)
Body Parameter — Request body with details to create store backup

Return type

{
  "lifecycleState" : "aeiou",

  "resourceId" : "aeiou",
  "resourceUrl" : "aeiou",

  "opcWorkRequestId" : "aeiou"
